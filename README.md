verview

Enterprise goods transportation management platform with protected staff dashboard, anonymous customer order submission with tracking codes, and GPS-based live shipment tracking with static map display.

Key Features

Staff login with role-based access (admin, dispatcher, driver)

Customer order submission with unique tracking codes

Shipment creation, assignment, and status management

Driver and vehicle registry management

GPS tracking via browser geolocation + static map

Shipment timeline with notes and status history

Public customer tracking page (no login required)

Responsive layout for desktop and mobile
